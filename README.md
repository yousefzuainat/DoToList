# DoToList

DoToList is a simple, lightweight to-do list web application that helps you manage your daily tasks. It is built using HTML, CSS, and pure JavaScript.

## Features

- **Add Tasks**: Easily add new tasks to your list.
- **Mark as Completed**: Click on any task to mark it as done or undo the completion status.
- **Delete Tasks**: Remove tasks you no longer need.
- **Local Storage**: Your tasks are saved locally in your browser, so you never lose them even if you refresh or close the page.

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Setup

Simply open the `index.html` file in your preferred web browser to start using the app. No additional installation or setup is required.
